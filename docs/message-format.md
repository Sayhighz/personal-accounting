# 📖 รูปแบบข้อความที่ระบบรองรับ (Message Format)

[cite_start]ระบบถูกออกแบบมาให้เข้าใจภาษาธรรมชาติผ่านการทำงานของ Regex และ AI [cite: 13, 3]

### 1. การบันทึกรายการ (Intent: Record)
[cite_start]ระบบจะตรวจจับ "ตัวเลข" ในข้อความเพื่อเริ่มกระบวนการบันทึก [cite: 13]
* **รูปแบบ:** `[ชื่อรายการ] [จำนวนเงิน]`
* **ตัวอย่าง:**
  * [cite_start]`ข้าวมันไก่ 45` [cite: 15]
  * [cite_start]`เงินเดือน 20000` [cite: 19]
  * [cite_start]`ค่ารถ 40 บาท` (ใส่คำว่าบาทหรือไม่ใส่ก็ได้) [cite: 17, 21]

### 2. การเรียกดูสรุป (Intent: Summary)
[cite_start]พิมพ์คำสั้นๆ เพื่อให้ระบบประมวลผลยอดรวม [cite: 23]
* [cite_start]**คำสั่ง:** `สรุป` [cite: 24]
* [cite_start]**สิ่งที่ระบบจะทำ:** อ่านข้อมูลทั้งหมดใน Google Sheets เฉพาะของ User คนนั้น แล้วนำมาคำนวณแยกประเภท [cite: 32]

### 3. กรณีที่ระบบไม่เข้าใจ (Unknown Intent)
หากข้อความไม่มีตัวเลขและไม่ใช่คำว่า "สรุป" ระบบจะส่งข้อความแนะนำการใช้งานกลับไปโดยอัตโนมัติ เพื่อช่วยเหลือผู้ใช้

### 4. การจัดการโดย AI
[cite_start]ระบบใช้โมเดล `ui-tars-1.5-7b` ในการวิเคราะห์หมวดหมู่ [cite: 9]
* [cite_start]**ตัวอย่าง:** หากพิมพ์ "กาแฟ 60" AI จะวิเคราะห์ว่าเป็น `รายจ่าย` และอยู่ในหมวดหมู่ `อาหาร` โดยอัตโนมัติ [cite: 13, 32]
