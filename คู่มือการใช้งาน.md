# 📊 คู่มือการใช้งาน Realtime Income/Expense Tracker

> ระบบบันทึกรายรับ-รายจ่ายผ่าน LINE อัตโนมัติ เชื่อมต่อกับ Google Sheets แบบ Real-time

---

## 🔧 ภาพรวมระบบ

ระบบนี้สร้างด้วย **n8n** โดยรับข้อความจาก LINE แล้วประมวลผลอัตโนมัติผ่านขั้นตอนดังนี้

```
LINE Message → วิเคราะห์ความตั้งใจ → AI จัดหมวดหมู่ → บันทึก Google Sheets → ตอบกลับ LINE
```

### องค์ประกอบหลัก

| ส่วนประกอบ | รายละเอียด |
|---|---|
| **LINE Messaging API** | รับ-ส่งข้อความกับผู้ใช้ |
| **n8n Workflow** | ประมวลผลและควบคุมลำดับงาน |
| **AI Agent (OpenRouter)** | วิเคราะห์และจัดหมวดหมู่รายการ |
| **Google Sheets** | เก็บข้อมูลรายการทั้งหมด |

---

## 💬 วิธีใช้งานผ่าน LINE

### 1. บันทึกรายการ

พิมพ์ **ชื่อรายการ + จำนวนเงิน** ลงใน LINE ได้เลย ระบบจะแยกแยะรายรับ/รายจ่าย และจัดหมวดหมู่ให้อัตโนมัติ

**ตัวอย่างรายจ่าย:**
```
ข้าวมันไก่ 45
ค่าไฟ 800
ค่าเดินทาง BTS 44 บาท
ซื้อยา 120
```

**ตัวอย่างรายรับ:**
```
รับเงินเดือน 15000
ขายของออนไลน์ 350
โบนัส 5000
```

> 💡 **เคล็ดลับ:** ใส่หรือไม่ใส่คำว่า "บาท" ก็ได้ ระบบอ่านได้ทั้งคู่

**ข้อความตอบกลับเมื่อบันทึกสำเร็จ:**
```
บันทึกรายจ่าย 45 บาท
รายการ: ข้าวมันไก่
หมวดหมู่: อาหาร ✅
```

---

### 2. ดูสรุปรายการ

พิมพ์ว่า **`สรุป`** เพื่อดูภาพรวมรายรับ-รายจ่ายทั้งหมดของคุณ

```
สรุป
```

**ตัวอย่างข้อความตอบกลับ:**
```
📊 สรุปรายการทั้งหมดของคุณ
(12 รายการ)

💰 รายรับ: 15,000 บาท
  • เงินเดือน: 15,000 บาท

💸 รายจ่าย: 3,240 บาท
  • อาหาร: 1,200 บาท
  • เดินทาง: 880 บาท
  • ค่าน้ำค่าไฟ: 800 บาท
  • สุขภาพ: 360 บาท

✅ ยอดคงเหลือ: 11,760 บาท
```

> ⚠️ หากยอดคงเหลือติดลบ จะแสดงเป็น ⚠️ แทน ✅

## 📂 หมวดหมู่ที่ระบบรองรับ

### หมวดหมู่รายจ่าย

| หมวดหมู่ | ตัวอย่างรายการ |
|---|---|
| อาหาร | ข้าว, กาแฟ, อาหารเย็น |
| เดินทาง | ค่า BTS, น้ำมัน, Grab |
| ที่พัก | ค่าเช่า, โรงแรม |
| สุขภาพ | ค่ายา, หมอ, ฟิตเนส |
| บันเทิง | หนัง, เกม, ท่องเที่ยว |
| ช้อปปิ้ง | เสื้อผ้า, ของใช้ |
| ค่าน้ำค่าไฟ | ค่าไฟ, ค่าน้ำ, อินเทอร์เน็ต |
| การศึกษา | หนังสือ, คอร์สเรียน |
| อื่น ๆ | รายการที่ไม่เข้าหมวดหมู่อื่น |

### หมวดหมู่รายรับ

| หมวดหมู่ | ตัวอย่างรายการ |
|---|---|
| เงินเดือน | เงินเดือน, ค่าจ้างประจำ |
| ขายสินค้า | ขายของ, รายได้จากธุรกิจ |
| ค่าจ้าง | งานฟรีแลนซ์, ค่าจ้างพิเศษ |
| โบนัส | โบนัส, ค่าตอบแทนพิเศษ |
| ดอกเบี้ย | ดอกเบี้ยธนาคาร |
| ลงทุน | กำไรหุ้น, กองทุน |
| อื่น ๆ | รายได้อื่น ๆ |

## 📋 โครงสร้างข้อมูลใน Google Sheets

ระบบบันทึกข้อมูลลงชีท **Transactions** โดยมีคอลัมน์ดังนี้

| คอลัมน์ | คำอธิบาย | ตัวอย่าง |
|---|---|---|
| Date | วันที่บันทึก (รูปแบบไทย) | 17/04/2568 |
| Time | เวลาบันทึก | 14:30 |
| Type | ประเภท | รายรับ / รายจ่าย |
| Amount | จำนวนเงิน | 45 |
| Description | รายละเอียด | ข้าวมันไก่ |
| Category | หมวดหมู่ | อาหาร |
| UserID | รหัสผู้ใช้ LINE | Uxxxxxxxx |

> 📌 ข้อมูลสรุปจะคำนวณจากรายการ **เฉพาะของผู้ใช้แต่ละคน** โดยใช้ UserID เป็นตัวแยก

## ❓ กรณีที่ระบบไม่เข้าใจคำสั่ง

หากพิมพ์ข้อความที่ไม่มีตัวเลข และไม่ใช่คำว่า "สรุป" ระบบจะตอบกลับคำแนะนำ:

```
ไม่เข้าใจคำสั่ง "..."

📝 วิธีใช้งาน:
• บันทึกรายการ → พิมพ์รายการพร้อมจำนวนเงิน
  เช่น: ข้าวมันไก่ 45, ค่าไฟ 800, รับเงินเดือน 15000
• ดูสรุป → พิมพ์ว่า สรุป
```

---

## ⚙️ สรุปคำสั่งที่ใช้ได้

| คำสั่ง | รูปแบบ | ตัวอย่าง |
|---|---|---|
| บันทึกรายการ | `[ชื่อรายการ] [จำนวนเงิน]` | `ข้าวผัด 60` |
| ดูสรุป | `สรุป` | `สรุป` |
