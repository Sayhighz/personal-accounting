# ชื่อโปรเจค: Automated Income & Expense Tracker

## Problem Statement

**WHO ใครเดือดร้อน?**
ผู้ใช้งานทั่วไปและธุรกิจขนาดเล็กที่ต้องการบริหารจัดการการเงินส่วนตัวหรือของธุรกิจ

**WHAT ปัญหาคืออะไร?**
การบันทึกรายรับ-รายจ่ายในปัจจุบันยังพึ่งพาการจดบันทึกด้วยมือหรือแอปพลิเคชันที่ซับซ้อน ส่งผลให้ผู้ใช้มักลืมบันทึกรายการ ข้อมูลผิดพลาด หรือสูญหาย และขาดรายงานสรุปที่เข้าถึงได้ง่าย

**WHEN เกิดบ่อยแค่ไหน?**
เกิดขึ้นทุกวันที่มีการใช้จ่ายหรือรับเงิน ซึ่งเป็นกิจกรรมประจำวันของทุกคน

**HOW MUCH เสียเวลา/เงินเท่าไหร่?**
เสียเวลาหลายนาทีต่อรายการในการเปิดแอปและกรอกข้อมูล รวมถึงความเสี่ยงในการสูญเสียเงินจากการบริหารงบประมาณที่ไม่มีประสิทธิภาพ

---

## Proposed Solution

สร้าง **Automated Income & Expense Tracker** ที่ให้ผู้ใช้บันทึกรายรับ-รายจ่ายผ่าน LINE เพื่อแก้ปัญหาการบันทึกข้อมูลที่ยุ่งยากและขาดรายงานสรุปอัตโนมัติ

---

## n8n Workflow Concept

**Trigger:** Webhook (รับข้อความจาก LINE Messaging API)

**Process:** IF Node แยกประเภทรายการ (รายรับ / รายจ่าย) → Code Node ประมวลผลและจัดรูปแบบข้อมูล

**Action:** เรียก Google Sheets API เพื่อบันทึกรายการ และ Scheduled Trigger ทุกต้นเดือนเพื่อสร้างรายงานสรุป

**Notify:** LINE (ส่ง Reply ยืนยันการบันทึก และรายงานสรุปรายเดือน)
