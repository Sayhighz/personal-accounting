## บทคัดย่อ

โครงงาน "ระบบบันทึกรายรับ-รายจ่ายอัตโนมัติ" มีจุดประสงค์เพื่อพัฒนาระบบอัตโนมัติสำหรับการจดบันทึกรายรับและรายจ่ายผ่านแอปพลิเคชัน Line โดยใช้ N8N เป็น Automation Workflow Engine ข้อมูลที่บันทึกจะถูกส่งตรงไปยัง Google Sheet แบบอัตโนมัติ และระบบจะสรุปรายงานรายรับ-รายจ่ายให้ทุกเดือนโดยอัตโนมัติ เพื่อช่วยให้การบริหารจัดการการเงินเป็นเรื่องง่ายและแม่นยำ

---

## 1.1 ความเป็นมาและความสำคัญของปัญหา

ในปัจจุบันการบริหารจัดการรายรับและรายจ่ายส่วนบุคคลหรือธุรกิจขนาดเล็กยังคงพึ่งพาการจดบันทึกด้วยมือหรือแอปพลิเคชันที่ซับซ้อน ทำให้ผู้ใช้งานเกิดความไม่สะดวกและมักละเลยการบันทึก จากการสำรวจพบว่า ปัญหาหลักที่พบได้แก่

- การจดบันทึกด้วยมือทำให้เกิดความผิดพลาดและข้อมูลสูญหาย
- ผู้ใช้ลืมบันทึกรายการ เนื่องจากต้องเปิดแอปแยกต่างหาก
- ขาดการสรุปรายงานรายเดือนที่ชัดเจนและเข้าถึงได้ง่าย
- ไม่มีระบบแจ้งเตือนหรือติดตามยอดรายรับ-รายจ่ายแบบเรียลไทม์

จากปัญหาดังกล่าว จึงเกิดแนวคิดในการพัฒนา **ระบบบันทึกรายรับ-รายจ่ายผ่าน Line และ N8N** ซึ่งเป็นระบบที่ผู้ใช้สามารถส่งข้อความผ่าน Line เพื่อบันทึกรายการได้ทันที โดยมีเป้าหมายเพื่อ

- ลดขั้นตอนการบันทึกข้อมูลให้สะดวกที่สุด โดยใช้ Line ที่คุ้นเคยอยู่แล้ว
- บันทึกข้อมูลลง Google Sheet แบบอัตโนมัติไม่ต้องกรอกเอง
- สร้างรายงานสรุปรายเดือนโดยอัตโนมัติเพื่อให้เห็นภาพรวมการเงิน
- ลดโอกาสที่ข้อมูลจะสูญหายหรือผิดพลาด

---

## 1.2 วัตถุประสงค์

1. เพื่อพัฒนา **ระบบรับข้อมูลรายรับ-รายจ่ายผ่าน Line Messaging** ที่ใช้งานสะดวกและรวดเร็ว
2. เพื่อสร้าง **Automation Workflow ด้วย N8N** ที่รับ Trigger จาก Line และบันทึกข้อมูลลง Google Sheet อัตโนมัติ
3. เพื่อพัฒนา **ระบบสรุปรายงานรายเดือน** ที่แสดงยอดรายรับ รายจ่าย และยอดคงเหลือใน Google Sheet
4. เพื่อสร้าง **Scheduled Trigger** ที่ทำงานสรุปรายงานทุกสิ้นเดือนโดยอัตโนมัติ
5. เพื่อลดภาระและเพิ่มความแม่นยำในการบริหารจัดการการเงินส่วนบุคคลหรือธุรกิจขนาดเล็ก

## 1.3 ขอบเขตของโครงงาน

### 1.3.1 การบันทึกรายรับ-รายจ่ายผ่าน Line

- ผู้ใช้ส่งข้อความผ่าน Line เพื่อแจ้งรายรับหรือรายจ่าย เช่น `รายรับ 500 ขายสินค้า` หรือ `รายจ่าย 200 ค่าน้ำ`
- ระบบรับข้อมูลจาก Line Webhook และส่งต่อไปยัง N8N Workflow
- รองรับการบันทึกหมวดหมู่รายการ เช่น อาหาร, เดินทาง, สาธารณูปโภค, รายได้จากงาน เป็นต้น

### 1.3.2 Automation ด้วย N8N

- ใช้ **N8N** เป็น Workflow Automation Engine หลัก
- รับ Trigger จาก Line Messaging API ผ่าน Webhook Node
- ประมวลผลและแยกประเภทข้อมูล (รายรับ/รายจ่าย, จำนวนเงิน, หมวดหมู่, วันที่)
- บันทึกข้อมูลลง Google Sheet อัตโนมัติผ่าน Google Sheets Node

### 1.3.3 การบันทึกข้อมูลลง Google Sheet

- สร้าง Google Sheet สำหรับเก็บข้อมูลรายรับ-รายจ่ายรายวัน
- บันทึกข้อมูล ได้แก่ วันที่-เวลา, ประเภท (รายรับ/รายจ่าย), หมวดหมู่, จำนวนเงิน, หมายเหตุ
- ข้อมูลถูกเพิ่มแบบ Append ต่อท้ายแถวอัตโนมัติทุกครั้งที่มีการบันทึก

### 1.3.4 รายงานสรุปรายเดือน

- N8N ทำงานสรุปรายงานรายเดือนอัตโนมัติผ่าน **Scheduled Trigger** (ทุกวันที่ 1 ของเดือน)
- สรุปยอดรายรับ รายจ่าย และยอดคงเหลือของเดือนที่ผ่านมา
- บันทึกผลสรุปลงใน Sheet แยกต่างหากภายใน Google Sheet เดิม
- ส่งข้อความสรุปรายงานกลับไปหาผู้ใช้ผ่าน Line อัตโนมัติ

## 2.1 การวิเคราะห์และออกแบบระบบ

ระบบ **Automated Income & Expense Tracker** เป็นระบบ Automation ที่พัฒนาขึ้นด้วย N8N เพื่อรองรับการบันทึกรายรับ-รายจ่ายผ่าน Line Messaging API อย่างครบวงจร โดยเน้นประสบการณ์ใช้งานที่ง่ายสำหรับผู้ใช้ทั่วไป และลดขั้นตอน Manual ในการบริหารจัดการการเงิน

### การวิเคราะห์ระบบ (System Analysis)

ก่อนพัฒนาระบบนี้ ผู้ใช้ส่วนใหญ่ยังพึ่งพาการจดบันทึกด้วยมือหรือ Spreadsheet ทั่วไปที่ต้องกรอกเอง ทำให้เกิดปัญหา เช่น

- ลืมบันทึกรายการเนื่องจากต้องเปิดแอปแยกและกรอกข้อมูลหลายขั้นตอน
- ข้อมูลสูญหายหรือกระจัดกระจายเมื่อเปลี่ยนอุปกรณ์
- ไม่มีรายงานสรุปรายเดือนที่สร้างขึ้นอัตโนมัติ

ระบบนี้จึงออกแบบมาเพื่อแก้ปัญหาเหล่านี้ โดยมีส่วนประกอบหลัก 3 ส่วน คือ **Line (Input Channel), N8N (Automation Engine), และ Google Sheets (Database & Report)**

### สถาปัตยกรรมระบบ (System Architecture)

```
[ผู้ใช้] 
    → ส่งข้อความผ่าน Line
    → [Line Messaging API] Webhook
    → [N8N Workflow]
        → แยกประเภท / ประมวลผลข้อมูล
        → บันทึกลง [Google Sheets] (รายการรายวัน)
        → ส่ง Reply กลับผู้ใช้ผ่าน Line

[N8N Scheduled Trigger] (ทุกวันที่ 1 ของเดือน)
    → ดึงข้อมูลจาก Google Sheets
    → คำนวณยอดรายรับ / รายจ่าย / คงเหลือ
    → เขียนรายงานลง Sheet สรุปรายเดือน
    → ส่งรายงานสรุปผ่าน Line
```

## 2.2 ทฤษฎีที่เกี่ยวข้อง / งานวิจัยที่เกี่ยวข้อง

1. **Workflow Automation (N8N)**
   - N8N เป็น Open-source Workflow Automation Tool ที่รองรับการเชื่อมต่อ API ต่าง ๆ ผ่าน Node-based Interface รองรับ Webhook Trigger, Scheduled Trigger, HTTP Request และ Google Sheets Node

2. **Line Messaging API**
   - Line Messaging API รองรับการรับ-ส่งข้อความแบบ Webhook ทำให้ระบบสามารถรับข้อมูลจากผู้ใช้และตอบกลับอัตโนมัติได้ผ่าน Reply Token

3. **Google Sheets as a Database**
   - Google Sheets สามารถใช้เป็น Lightweight Database สำหรับโครงงานขนาดเล็ก รองรับการ Append Row, Read Range และ Update Cell ผ่าน Google Sheets API ที่ N8N มี Node รองรับพร้อม

4. **Scheduled Automation & Report Generation**
   - การใช้ Scheduled Trigger ใน N8N เพื่อรัน Workflow สรุปรายงานอัตโนมัติรายเดือน ช่วยลดขั้นตอน Manual และทำให้รายงานมีความสม่ำเสมอ

5. **งานวิจัยที่เกี่ยวข้อง**
   - งานวิจัยด้านระบบบันทึกการเงินส่วนบุคคลด้วย Chatbot
   - งานวิจัยด้านการประยุกต์ใช้ Low-code / No-code Automation ในธุรกิจขนาดเล็ก
